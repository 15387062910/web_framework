## 密码加密及重置功能演示程序
### 程序功能
    1. 实现登录注册功能
    2. 实现密码加密存储(加盐加密)
    3. 实现修改密码(重置)
    
### 程序结构
    data        存储用户信息
    routes      路由程序
    static      静态文件(CSS 图片)
    templates   模板文件(HTML)
    models.py   数据存储相关类定义
    server.py   服务器主程序
    utils.py    功能函数文件
    
### 详细说明与结束: 见目录下的密码加密及重置程序说明.py
